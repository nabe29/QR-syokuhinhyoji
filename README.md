# QR食品表示
キッチンカーや屋台などで、食品表示がなく、情報が不明瞭なお店で食べ物を買ったとき、材料やアレルギー情報をより詳しく知ることが出来るアプリです。
<br>
メンバー：3人
<br>
ProtoPedia：https://protopedia.net/prototype/5781

## 使用技術
+ Firebase Cloud Firestore
+ Firebase Hosting

## 担当箇所
+ 全体の画面設計
+ UI
+ ログイン機能
+ PWA
